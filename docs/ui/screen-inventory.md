# Screen Inventory

## 核心页面
- OnboardingScreen: 首次启动与连接引导。
- ChatListScreen: 会话列表。
- ChatDetailScreen: 消息流、输入框、引用和操作。
- KnowledgeScreen: 数据集列表与入口。
- DatasetDetailScreen: Collection、Chunk、检索测试。
- AppSelectorScreen: 应用选择与详情入口。
- SettingsScreen: 各类配置总览。

## 关键 ViewModel
- ChatListViewModel
- ChatDetailViewModel
- KnowledgeViewModel
- DatasetDetailViewModel
- SettingsViewModel
