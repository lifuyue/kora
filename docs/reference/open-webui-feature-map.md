# Open WebUI Feature Map

## 目标
把 Open WebUI 的高频能力映射到 Kora 的 Android 端实现优先级，避免盲目复制桌面端复杂性。

## 高优先映射
- 流式聊天 -> `features/chat/streaming-chat.md`
- Markdown 与代码块 -> `features/chat/markdown-rendering.md`
- 会话管理 -> `features/conversations/`
- 引用来源 -> `features/chat/citations.md`
- 多应用切换 -> `features/apps/app-selector.md`

## 选择性映射
- 消息反馈
- 推荐问题
- 多模态输入
- 语音能力

## 暂缓项
- 高度插件化桌面配置
- 复杂管理后台能力
